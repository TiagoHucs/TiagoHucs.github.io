Curso Java – Aventura Espacial

Configuração do Ambiente

1. Baixando o IntelliJ IDEA Community Edition

O IntelliJ IDEA é o IDE que vamos usar para escrever e executar nossos códigos Java. Siga os passos abaixo:

  1. Acesse o site oficial do IntelliJ IDEA: https://www.jetbrains.com/idea/download/
  2. Escolha a versão Community Edition (gratuita) e clique em "Download".
  3. Após o download, execute o instalador e siga as instruções da instalação.

2. Instalando o JDK (Java Development Kit)

O IntelliJ IDEA precisa do JDK para compilar e executar programas Java.

  1. Baixe o JDK mais recente (por exemplo, JDK 17).
  2. Instale o JDK seguindo as instruções do instalador.
  3. Anote o caminho de instalação, pois pode ser necessário configurar no IntelliJ.

3. Criando o Primeiro Projeto Java

  1. Abra o IntelliJ IDEA.
  2. Selecione New Project.
  3. Escolha Java e verifique se o JDK está selecionado.
  4. Dê um nome ao projeto, por exemplo: AventuraEspacial.
  5. Clique em Finish para criar o projeto.

4. Criando a Classe Main

Vamos criar a classe principal onde escreveremos nossos códigos de treinamento:

  1. Clique com o botão direito na pasta src → New → Java Class.
  2. Digite o nome da classe: Main.
  3. Clique em OK.

Seu arquivo Main.java deve conter o seguinte código inicial:

public class Main {
    public static void main(String[] args) {
        System.out.println("Bem-vindo à Aventura Espacial!");
    }
}

Agora seu ambiente está pronto e você pode seguir para os **exercícios da Parte 1 – Cadete**.

Vamos lá!
Dica: Sempre salve o projeto antes de executar. Use o botão ▶️ (Run) no IntelliJ para ver a saída no console.
